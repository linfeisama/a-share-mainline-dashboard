# A股主线选股看板

这是一个面向趋势选股的中文数据看板，用申万行业分层、概念交叉确认和公司基本面代理指标，集中展示市场主线与第一梯队候选。它只解决方向识别和公司身份筛选，不判断当前位置，也不提供收益承诺。

## 直接访问

- [打开公开在线看板](https://linfeisama.github.io/a-share-mainline-dashboard/)
- [查看手动更新任务](https://github.com/linfeisama/a-share-mainline-dashboard/actions/workflows/%E6%9B%B4%E6%96%B0%E5%9C%A8%E7%BA%BF%E7%9C%8B%E6%9D%BF.yml)

## 看板内容

- 市场状态，以及 31 个申万一级行业的完整横向比较
- 二级方向候选和叶子行业全覆盖计算
- 与申万行业分开评分的概念板块确认
- 主线内第一梯队与候选公司，股票代码限制为 `60`、`00` 开头
- 与个股分开列示的行业或主题 ETF 替代工具
- 个股名称直接链接 TradingView 图表

## 更新方式

在线站点不设置定时任务。点击看板头部“更新行情”会打开 GitHub Actions 页面；手动运行“更新A股主线在线看板”，等待发布完成后刷新看板即可。

本机使用时，双击 `启动看板网站.cmd`，再访问 `http://127.0.0.1:8765`。本地按钮会直接获取数据、重新计算并载入最新结果，程序不会自动打开浏览器。

更完整的运行和部署说明见 [在线部署说明.md](在线部署说明.md)。

## 风险边界

数据源可能存在延迟、缺失或口径变化。行业资金方向由相对收益、扩散度、持续性、趋势和量能代理，不等同于真实逐笔资金流。任何候选名单都需要结合公告、财报、政策与交易纪律人工复核。
