# 在线部署说明

## 公开地址

- 在线看板：https://linfeisama.github.io/a-share-mainline-dashboard/
- 公开仓库：https://github.com/linfeisama/a-share-mainline-dashboard

## 本机手动更新

1. 双击 `启动看板网站.cmd`，保持命令窗口运行。
2. 自己在浏览器访问 `http://127.0.0.1:8765`，程序不会自动打开浏览器。
3. 点击页面头部的“更新行情”，等待页面自动载入最新结果。
4. 不再使用网站时，在命令窗口按 `Ctrl+C` 停止服务。

直接打开静态 HTML 仍可查看最后一次结果，但静态文件本身不能启动 Python，因此按钮会提示先启动本地网站。

## 当前结构

- `运行一次.py`：批量更新市场数据并生成看板。
- `网站服务.py`：接收页面按钮请求并运行一次数据更新。
- `启动看板网站.cmd`：启动本机网站，不自动打开浏览器。
- `在线站点/index.html`：在线托管平台读取的默认首页。
- `在线站点/网站首页.html`：同内容的中文文件名副本。
- `.github\workflows\更新在线看板.yml`：仅支持手动运行或代码更新后发布，不包含定时任务。

## 数据更新含义

本地网站的“更新行情”按钮会调用本机分析程序，完成后自动载入新页面。GitHub Pages 属于静态托管，页面按钮不能在云端直接运行 Python；在线版点击“更新行情”会打开 GitHub Actions 手动更新入口。运行“更新A股主线在线看板”后，等待发布完成再刷新在线页面。项目不包含任何定时任务。

## 在线手动更新

1. 打开在线看板，点击头部“更新行情”。
2. 在打开的 GitHub Actions 页面点击 `Run workflow`，再确认运行。
3. 等待任务完成后回到在线看板，刷新页面查看最新结果。

仓库为公开仓库，GitHub Pages 发布来源为 GitHub Actions。
